# chzzk-map

치지직 스트리머 간의 유사도를 그래프로 시각화해서 스트리머간의 관계를 한눈에 볼 수 있습니다.

`d3.js` 의 Force-directed 그래프를 사용해서 유사도를 시각화했습니다.
